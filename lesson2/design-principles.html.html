Hick's Law

BYU pathway

Link to: BYU pathway

Here we can see how Hick's Law is perfectly applied, only 2 main options are being given to the user, one is to "Apply now" and the other is to "Request more info", by doing this we comply with the Hick's Law and do not distress the user with many options.

PARC: Repetition

Reddit

Link to: Reddit

Reddit's website is an example of repetition because it focuses on creating consistency by using the same position in all the elements on its home page. We can see that all posts have the same repeating pattern, a title, a right-aligned photo, the date of publication, the subreddit where the post was published, the likes, the comment counter and the awards. A visual pattern is created for all posts making the user experience more accessible, useful and intuitive.
